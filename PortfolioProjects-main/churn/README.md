Customer Churn Analysis Dashboard

Project Overview

This project focuses on analyzing customer churn to identify patterns and key factors influencing customer retention. Using Power BI and DAX, I built an interactive dashboard that highlights churn rates, customer demographics, and behavioral trends. The goal is to provide actionable insights for improving retention strategies and reducing revenue loss.

Tools & Skills Used
Power BI → Data visualization & dashboard design
DAX (Data Analysis Expressions) → Custom measures and calculations
Data Cleaning & Preparation → Ensured dataset accuracy and usability
Business Analytics → Focused on customer behavior and retention strategies
